# 更新日志 (CHANGELOG)

## v26.6.16 — 2026-06-16

### 🔒 安全修复

- **XSS 漏洞修复**：`escapeHtml()` 补全单引号转义（`'` → `&#39;`），堵住 HTML 属性注入风险
- **插件类型注入修复**：`ModuleRegistry.renderList()` 中 `m.type` 现在经过 `escapeHtml()` 转义后再拼入 CSS class 属性
- **日志图标注入修复**：`renderLogList()` 的 `icon` 字段现在经过 `escapeHtml()` 转义
- **模块导入文件大小限制**：新增 512KB 上限，防止恶意大文件导致浏览器卡死

### 🐛 Bug 修复

- **讲台视角门标签错误**：修复切换讲台视角后两个门都显示"前门"的问题，标签现在始终反映门的真实身份（"🚪 前门"/"🚪 后门"），不再交换
- **讲台视角门位置重叠**：修复切换视角后两个门被放到同一侧的问题，前门（近讲台）和后门（远讲台）现在分别位于正确的位置
- **讲台视角 hover 失效**：修复 `.seat:hover` 覆盖 `scaleY(-1)` 导致讲台视角下鼠标悬停时座位翻转失效的 CSS 问题
- **讲台视角状态不持久化**：修复刷新页面后讲台视角状态丢失的问题，现在通过 `saveConfig()` 保存到 localStorage
- **讲台视角 UI 不同步**：修复切换视角后"⋯ 更多"菜单和"👁️ 视图"下拉菜单中的文字不更新的问题，新增 `updateViewModeUI()` 统一同步所有入口
- **配置恢复嵌套对象丢失**：修复初始配置恢复时 `settings` 浅合并导致 `weights`/`quickInfoItems` 等嵌套对象被整体替换丢失默认值的问题
- **插件函数方法丢失**：修复 `saveConfig()` 序列化插件时 `JSON.stringify` 静默丢弃函数导致插件 `init`/`beforeDraw`/`afterDraw` 等方法刷新后失效的问题
- **`enableSeat()` 缺少通道检查**：修复启用座位后未调用 `checkAisles()` 导致过道状态不更新的问题
- **`enableColumn()` 缺少通道检查**：修复启用列的 redo 回调中缺少 `checkAisles()` 调用
- **热力图类型不持久化**：修复 `heatmapType` 未保存到 localStorage 导致刷新后重置为默认值
- **帮扶链成绩筛选错误**：修复 `buildTutoringChain()` 中用 `||` 合并分数导致 0 分被错误回退的问题，改用 `??`
- **关系影响距离计算错误**：修复 `applyRelationships()` 使用曼哈顿距离而非有效距离导致关系加分在有过道阻隔时仍生效的逻辑不一致
- **右键菜单冲突**：修复连续打开座位右键菜单和列右键菜单时第一个无法通过点击外部关闭的问题，现在两个菜单互斥关闭
- **`poolSubtitle` 空指针**：修复 `updateStats()` 中 `poolSubtitle` 缺少 null 保护导致元素缺失时崩溃
- **`mmItem.lastChild` 空指针**：修复 `updateViewModeUI()` 中 `lastChild` 可能为文本节点或 null 导致切换视角时崩溃
- **拼音表重复键**：移除 `_pinyinMap` 中 `干`和`郏` 的重复定义
- **阈值不一致**：统一 `explainPairing()` 和 `peerInfluence()` 的成绩差阈值（15 分）
- **0 分与无数据歧义**：修复 `getScore()` 中 `student.score || 0` 无法区分 0 分和无数据，改用 `??`

### ⚡ 性能优化

- **CDN 脚本加载优化**：所有 CDN 脚本和应用脚本添加 `defer`，不再阻塞首屏渲染
- **Toast 数量限制**：添加 5 个上限，超出时自动移除最旧的 toast，防止快速触发时 DOM 堆积
- **`updateStats()` DOM 查询优化**：提取 `safe()` 辅助函数，每个元素只调用一次 `getElementById`
- **平台座位查找去重**：新增 `getSeatElement()` 工具方法，替换全文 7 处重复的平台座位元素查找逻辑
- **Monte Carlo Worker 泄漏防护**：新模拟启动前自动终止旧 Worker，防止内存泄漏

### 🧹 代码质量

- **删除死代码**：移除 `ui.js` 中未使用的 `ranges`/`dist` 变量和 `studentIds` 变量
- **删除不可达代码**：移除 `drawStudent()` 中永远无法执行的 `shift()` 回退
- **移除内联事件处理器**：6 处 HTML `onclick` 属性全部移除，改为 `addEventListener` 绑定，符合关注点分离原则
- **允许用户缩放**：移除 `user-scalable=no`，修复 WCAG 1.4.4 无障碍性违规

### 🆕 新增功能

- **视图下拉菜单添加讲台视角选项**：在"👁️ 视图"下拉菜单中新增 `🎓 讲台视角` 选项，与左上角按钮和"⋯ 更多"菜单共 3 个入口，切换后全部同步更新
- **讲台视角操作日志**：切换讲台视角现在会记录到操作日志中
