📖 操作指南

📋 功能索引

🚀 3步开始

  1. 设置布局：切换到"布局"Tab → 设置排数、列数
  2. 导入学生：切换到"学生"Tab → 粘贴名单或导入Excel
  3. 开始抽取：点击底部 🎲 抽取下一个 或 ⚡ 一键自动抽取

🧠 智能推荐 & 自定义算法

  • 点击底部 🧠 智能推荐 ▾ 按钮，下拉菜单有两个选项：
    • 🧠 智能推荐：分析当前座位，找出可优化的互换方案
    • 🧬 自定义算法：导入你自己的推荐算法（.js文件）
  • 推荐面板以侧边栏形式展示，鼠标悬停可预览高亮效果
  • 自定义算法需实现 peerInfluence 接口，详见"AI算法开发"Tab

📋 黑白名单特殊标记

  • 在名单中使用 * 前缀标记"锚点"成员，例如：*张三 李四 王五
  • 黑名单：默认只有组内第一个被抽到的人作为锚点，其他人只在靠近锚点时被惩罚
  • 白名单：默认所有已抽到的组员都可吸引剩余组员靠近
  • 使用 * 标记可指定哪些人是锚点（惩罚/吸引只对标记成员生效）

🪑 座位操作

  • 拖拽互换：按住座位拖到另一个座位上松开
  • 点击互换：先点第一个座位（蓝色边框），再点第二个
  • 右键菜单：右键/长按座位进行更多操作
  • 切换午休：点击已有人的座位切换午休状态
  • 批量模式：按 B 键进入，点选多个座位后批量操作

⌨️ 快捷键

  • 空格：抽取下一个学生
  • ESC：取消选取/关闭弹窗/退出批量模式
  • Ctrl+S：手动保存 | Ctrl+Z：撤销 | Ctrl+Y：重做
  • B：批量选择模式 | 方向键：座位间导航

📦 模块 & 插件

  • 插件管理（高级Tab）：导入 .js 插件扩展功能，支持安全扫描
  • 模块管理（系统Tab）：查看系统模块状态，可导入自定义模块
  • 自动抽取间隔：在高级Tab → 插件管理区域可直接调整

💡 导入提示
  • Excel 导入时，系统会自动匹配表头中的科目名称（如"语文""数学"）
  • 如果 Excel 没有表头，按默认列顺序解析：姓名、性别、午休、各科成绩…
  • 成绩范围按科目满分自动修正（可在"导入学生→科目管理"中设置各科满分）
  • 性格支持：外向、内向、中性
  • 职务支持：班长、副班长、学习委员、体育委员、文艺委员、劳动委员、小组长、课代表

🧮 综合评价指数

综合评价指数是一个 0-100 的分数，综合考量学生的学业成绩、性格特征、兴趣爱好、班级职务等多维度信息，是智能座位编排的核心依据。

综合评价 = Σ(维度得分 × 维度权重) / Σ(有效权重) 学业成绩 = 所有科目成绩的平均值 性格评分 = 外向70 / 内向65 / 中性80（适应性评分） 爱好评分 = min(100, 50 + 爱好数量×10) 职务评分 = 班长95 / 学习委员90 / 体育委员85 / ...

🔮 抽取概率算法

公平可预测：每位学生有基础概率，经黑白名单、性别均衡、历史邻座惩罚调整后归一化。
公平不可预测：在可预测基础上加入随机扰动，使结果更难预测但仍然公平。

🧠 智能推荐逻辑

系统分析已排座位中所有相邻学生对的"良性影响分"，找出可以优化的互换方案：

良性影响分 = 学业互补(0-20) + 性格互补(0-15) + 共同爱好(0-20) + 职务多样(0-10) + 性别均衡(0-5) 1. 计算当前所有相邻对的总良性影响分 2. 尝试每对学生互换，计算互换后的总分变化 3. 选择总分提升最大的互换方案作为推荐 4. 每条推荐附带具体原因说明

🔥 热力图类型

  • 综合评价热力图：基于上述多维评价指数，颜色越绿表示综合能力越强
  • 综合成绩热力图：仅基于所有科目成绩平均值
  • 单科成绩热力图：基于选定科目的单科成绩

🧬 AI 推荐算法开发指南

本文档供开发者参考，用于创建自定义的座位推荐算法。自定义算法可替换或增强内置的智能推荐引擎。
算法接口

自定义算法需实现一个 JavaScript 对象，包含以下核心方法：

const CustomAlgorithm = { name: "我的算法", version: "1.0.0", description: "自定义推荐算法", // 核心方法1：计算两名学生的良性影响分 peerInfluence(s1, s2, context) { // s1, s2: 学生对象 // context: { seats, settings, rows, cols } // 返回: number (越高表示越应该坐在一起) return 0; }, // 核心方法2：生成互换推荐 generateRecommendations(seatedStudents, context) { // seatedStudents: [{student, seat}, ...] // 返回: [{seat1, seat2, reason, priority}] return []; }, // 可选：计算学生综合评分 (用于热力图) getScore(student) { return 50; } };
学生对象结构
{ id: Number, // 唯一ID name: String, // 姓名 gender: 'male'|'female', lunch: Boolean, // 是否午休 pinned: Boolean, // 是否固定 scores: { // 多科成绩 '语文': 85, '数学': 92, '英语': 78 ... }, personality: '外向'|'内向'|'中性'|null, hobbies: ['篮球', '绘画'], position: '班长'|'学习委员'|null }
上下文对象
context = { seats: Array, // 所有座位对象数组 rows: Number, // 排数 cols: Number, // 列数 settings: Object, // 系统配置 blacklist: Array, // 黑名单 whitelist: Array // 白名单 }
内置维度参考
  • 学业互补：成绩差异 15-40 分为最佳帮扶区间 (+20分)
  • 性格互补：外向+内向搭配 (+15分)，中性适配 (+8分)
  • 共同爱好：每个共同爱好 +8分，上限 20分
  • 职务多样：不同职务搭配 +10分
  • 性别均衡：男女搭配 +5分
  • 行内成绩均衡：同排成绩方差大 +10分
  • 午休聚集惩罚：相邻午休过多 -10分
导入与使用
  • 点击底部"智能推荐"旁的下拉菜单，选择"自定义算法"
  • 导入 .js 文件后，算法会自动注册并可切换使用
  • 支持同时导入多个算法，在面板中切换
  • 自定义算法经过安全扫描，禁止危险操作

🔌 插件开发指南

插件基本结构
const MyPlugin = { name: "插件名称", description: "描述", version: "1.0.0", defaultEnabled: true, hasSettings: false, defaultSettings: {}, init() { /* 初始化 */ }, beforeDraw(availableStudents, probabilities, nextSeat) { // 抽取前调用，可修改概率 return { availableStudents, probabilities }; }, afterDraw(student, seat) { /* 抽取后调用 */ }, beforeExport(data) { return data; } // 导出前调用 }; PluginManager.register('my-plugin', MyPlugin);

  • Toast.success(msg) / Toast.error(msg)
  • addLog(icon, text) — 记录操作日志
