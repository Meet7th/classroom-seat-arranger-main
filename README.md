# 🎓 SeatWise Beta — 教室座位智能编排系统 Bate

> 本项目由 [Claude Code](https://claude.ai/code) 通过**氛围编程 (Vibe Coding)** 方式开发。

一款运行在浏览器中的教室座位编排工具，教师只需粘贴学生名单，系统即可自动分配座位。
